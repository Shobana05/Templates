Library Glossary

Definitions
Library 
      A collection of resources, such as books, periodicals, and multimedia materials, organized for use and
      maintained by a public  body, an institution, or a private individual.
Interlibrary Loan (ILL)
      Interlibrary loan is a service through which a patron of one library can borrow books, DVDs, or other materials
      owned by another library.
Catalog
      A database or listing of all the materials available in a library, including books, journals, multimedia,
      and other resources.
Circulation Desk 
      The service point in the library where patrons check out and return library materials.
Shelf location 
       The specific location within the library where a particular item is housed, typically identified by a
       call number or shelf marker.

Acronym

OPAC - Online Public Access Catalog

ISBN - International Standard Book Number
